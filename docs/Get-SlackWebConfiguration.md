# Get-SlackWebConfiguration

## SYNOPSIS
Retrieves your current configuration (token and d-cookie).

## SYNTAX

```
Get-SlackWebConfiguration [<CommonParameters>]
```

## DESCRIPTION
Retrieves your current configuration (token and d-cookie).

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-SlackWebConfiguration
```

Retrieves your current configuration (token and d-cookie).
